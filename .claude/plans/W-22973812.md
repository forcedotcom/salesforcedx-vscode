- CI orgE2E.yml: dev hub auth'd globally alias `hub` `--set-default-dev-hub`; runs all org desktop specs (auto-picks new spec). dev hub alias from `sf config get target-dev-hub --json`.
- Workspace `createTestWorkspace` (desktopWorkspace.ts) has no `config/*-scratch-def.json` → FileSelector finds 0 → cancels. Must add scratch-def to workspace fixture.
- Reuse: `selectQuickInputOptionByTyping`, `executeCommandWithCommandPalette`, `waitForNotification` (all exported, index.ts).
- No reload needed: `TargetOrgRef` watcher updates status bar on config change; both orgs globally auth'd so picker lists both.
