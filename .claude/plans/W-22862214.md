#### `packages/salesforcedx-vscode-core/src/context/workspaceOrgShape.ts` — rewrite

Pseudocode:
```
export const getOrgShape = async (_username: string): Promise<OrgShape> => {
  if (!workspaceUtils.hasRootWorkspace()) return 'Undefined';
  return getRuntime().runPromise(
    Effect.gen(function* () {
      const api = yield* (yield* ExtensionProviderService).getServicesApi;
      const ref = yield* api.services.TargetOrgRef();
      const info = yield* SubscriptionRef.get(ref);
      return info.isScratch ? 'Scratch'
        : info.isSandbox ? 'Sandbox'
        : (info.alias ?? info.username) ? 'Production'
        : 'Undefined';
    }).pipe(Effect.catchAll(() => Effect.succeed('Undefined' as OrgShape)))
  );
};
```
Notes:
- keep `hasRootWorkspace()` guard (preserves original semantics)
- `Effect.catchAll` so any service-extension load failure / runtime error degrades to `'Undefined'` (matches old behavior when no workspace)
- `_username` arg kept (einstein-gpt bridge compat), unused — prefix `_` to silence lint
- empty `{}` from `defaultOrgRef` → `isScratch`/`isSandbox` undefined → falls through to alias??username falsy → `'Undefined'` (verified path)
- drop imports: `isASandboxOrg`, `isAScratchOrg`, `getTargetOrgOrAlias`
