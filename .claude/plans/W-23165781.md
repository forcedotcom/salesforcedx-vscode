# W-23165781 ‚Äî apex-testing: drop telemetry shim + OUTPUT_CHANNEL proxy

## Context
2 activation-block cleanups, bundled so later WIs don't fight `index.ts`. Pkg: `salesforcedx-vscode-apex-testing`.

- Telemetry shim `src/telemetry/telemetry.ts` = backward-compat wrapper. Current shim is the fire-and-forget anti-pattern: `runPromise(...).catch(() => {})` (telemetry.ts:13-22) silently swallows errors ‚Äî FORBIDDEN (effect SKILL.md anti-patterns: swallowing errors / `.catch(() => {})`). Do NOT replicate it. Spans already emitted via `registerCommandWithRuntime` + `Effect.withSpan`. 3 refs:
  - `index.ts:233` `sendExtensionDeactivationEvent()` ‚Äî drop
  - `views/testController.ts:1356,1537` `sendEventData(...)` ‚Äî inline to direct span (preserve `apexTestRun`/`apexTestReportGenerated` events) via `getApexTestingRuntime().runFork(...)` (NOT runPromise+catch). `runFork` surfaces failures to the runtime error channel instead of swallowing them; pattern: `services/src/vscode/registerCommand.ts:66` `runtime.runFork(f().pipe(Effect.withSpan(...)))`. `getApexTestingRuntime()` returns a `ManagedRuntime` exposing `.runFork`.
- OUTPUT_CHANNEL Proxy in `channels/index.ts` ‚Äî sync handle for `.show()`. 3 callers, all Effect pipelines w/ `api`/`channelService` already in scope:
  - `commands/apexTestRun.ts:126` (channelService @100)
  - `commands/apexTestRunCodeAction.ts:82` (channelService @43)
  - `commands/apexTestSuite.ts:119` (channelService @106, inside `buildSuite`)
  - replace w/ `yield* channelService.showChannel` ‚Äî `showChannel` is a first-class `Effect` member of `ChannelService` (`services/src/vscode/channelService.ts:22` `showChannel: Effect.sync(() => channel.show())`). Use it directly; do NOT use `(yield* channelService.getChannel).show()` (skips the idiomatic API, two yields where one suffices).
- Stale jest mocks to clean: `test/jest/views/testController.test.ts:79` (telemetry mock), `test/jest/utils/testReportGenerator.test.ts:19` (OUTPUT_CHANNEL + nonexistent `channelService` export; src no longer uses)
