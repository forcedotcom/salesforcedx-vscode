# W-23167310 — close WI on merged only + extractPrUrl picks live (last) PR

## Context

- `auto-build-wi.js` monitor closes WI on PR state `merged` OR `closed`.
- Details__c may hold 2+ `<strong>PR:</strong>` markers (abandoned attempt appended first, live PR appended later). `extractPrUrl` (L286) returns the FIRST regex match → always the OLDEST/abandoned URL.
- Repro W-23125840: PR 7557 (abandoned, closed) appended first + 7561 (live) appended later. `extractPrUrl` → 7557. Monitor reads 7557 `closed` → WI set `Status__c=Closed`, no resolution. Live 7561 never examined.

### Two coupled defects (both must be fixed)

1. **Wrong URL wins (root cause).** First-match means the abandoned PR is the one checked everywhere `wi.prUrl` flows: monitor `checkPrStatePrompt` (L512 `gh pr view ${wi.prUrl}`), `closeMergedPrompt` (L561), candidate re-queue picker (L1276 `extractPrUrl(c.details)`). Gating close on `merged` alone does NOT fix this — it just turns "wrongly close WI" into "wait on the abandoned-closed PR forever; live PR never examined; WI never advances to finalize/close-wi."
2. **`closed` should not close the WI.** Even with the correct URL selected, a PR that is `closed`-not-merged is an abandoned attempt, never a completion. Only `merged` is a terminal success.

### Decision on which URL wins (the advocate's required call)
**Last `<strong>PR:</strong>` match wins.**
- workflow only APPENDS markers (reconcilePrPrompt L549 `existing + snippet`; PR-open persist L961-968) → last-appended = live PR; deterministic.
- scan-all + live-query-per-tick rejected: extra `gh` calls/tick; picker (L1276) lacks per-URL in-flight state machine.
- last-match = minimal correct change, benefits monitor + picker uniformly.

### Touch points (`.claude/workflows/auto-build-wi.js`)
- L282-289 `extractPrUrl`: first-match regex → last-match.
- L1079: `merged || closed` → `merged` only; `closed` → explicit `wait` + log.
- L561 `closeMergedPrompt`: drop `${r.prState.state}` interpolation (only `merged` reaches it).
- README.md L94, L197, L201: docs say merged/closed closes WI.
