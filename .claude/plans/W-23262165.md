# W-23262165 — Replace lwc utils-vscode readFile with FsService

## Context

- WI paths stale. Real usage: `lwcTestController.ts` in `testExplorer/` (not `testRunner/`), call at `readJestResults` (~line 670), not 516.
- Import: `lwcTestController.ts:7` `import { readFile } from '@salesforce/salesforcedx-utils-vscode'`.

- TEST REGRESSION RISK: existing test `lwcTestController.test.ts:637` mocks only `vscode.workspace.fs.readFile`. Verified by reproducing the Phase-1 edit + running the suite. → test MUST be updated (Phase 1 sub-task).
- `salesforcedx-vscode-services`: devDep (package.json:48) + extensionDependency; runtime API via `ExtensionProviderService`. Already wired in package.
